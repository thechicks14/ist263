About This Lab

In this lab, you will explore the website development lifecycle and create a style document to define your color and design choices. You will also practice building a sitemap for a fictitious museum called 263 Modern Art, and apply styling techniques such as hex colors, pseudo selectors, and box model properties.

Website Development Lifecycle

  1. Define Project - Gather information, understand audience and competitors, develop requirements.
  2. Planning - Create a project timeline, budget, gather media and copy.
  3. Design - Wireframe, sitemap, style document, mockup site design.
  4. Develop - Write HTML, CSS, and JavaScript, ensure SEO and accessibility.
  5. Test and Deploy - Test usability and functionality, launch the website.
  6. Maintain - Update and improve the site as needed.

Color Scheme

Choose your color palette using Color Hex. Record your hex colors in your style document for:

  • Body background
  • Text
  • Links
  • Headings
  • Navigation background
  • Navigation highlight
  • Navigation link color
